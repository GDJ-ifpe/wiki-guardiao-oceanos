# Sobre o Projeto
O "Guardião dos Oceanos" é um projeto de extensão que utiliza a criação de jogos digitais como ferramenta prática para sensibilizar e educar a comunidade acadêmica e escolar sobre os impactos das mudanças climáticas nos ecossistemas marinhos. 

Este repositório foi construído para ser o seu **guia de estudos e material de consulta**. Ao invés de apenas consumir conteúdo, nós construímos materiais para ensinar programação, design, arte digital e engenharia de som.

# Sobre o GDJ
Somos o grupo de estudos de desenvolvimento de jogos do IFPE campus Palmares. A proposta do grupo é criar uma comunidade contínua com a cultura de desenvolvimento de jogos, unindo interessados em se aprofundar nesse universo.
